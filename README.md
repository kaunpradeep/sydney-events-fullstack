Sydney Events Fullstack

This is a full-stack web application that scrapes public event data for Sydney, Australia and displays it in a clean, user-friendly interface.

The project demonstrates end-to-end development including web scraping, backend APIs, database storage, and frontend UI.

Features

Automatically scrapes events from public event websites

Stores event data in MongoDB

Displays events in a minimal UI

Each event shows:

Event name

Date & time

Venue / location

Short description

Source website

GET TICKETS button

Captures user email before redirecting to original event link

Tech Stack

Frontend: React

Backend: Node.js, Express

Database: MongoDB

Scraping: Python, BeautifulSoup

Version Control: Git, GitHub
