# Sydney Events Full Stack Assignment

This project scrapes public Sydney events and displays them using a full stack web application.

## Stack
- React
- Node.js / Express
- MongoDB
- Python (BeautifulSoup)

## Run Locally
1. Start MongoDB
2. Backend: cd backend && npm install && npm start
3. Scraper: cd scraper && python scrape_events.py
4. Frontend: cd frontend && npm install && npm start

Project runs locally for assignment submission.
